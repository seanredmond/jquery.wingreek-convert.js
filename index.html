jquery.wingreek-convert.js

What is it?

jquery.wingreek-convert.js is jQuery plugin for converting legacy WinGreek-encoded text into Unicode. If you had a large amount of WinGreek encoded text that you wanted to put on a web page, with this plugin you can paste it directly into your HTML and convert it automatically into Unicode.

WinGreek was a popular utility and font for composing Classical Greek from the days of Windows 3.1, created by Peter J. Gentry and Andrew M. Fountain. Due to its popularity, many pre-Unicode Greek fonts followed its mapping of characters.

jquery.wingreek-convert.js has evolved itself from an original Perl version of an online converter I created sometime around 1998.

How to get it

  • Download Version 1.0. Contains full (26K) and minified (6.6K) versions.
  • Fork the project from GitHub.

How to use it

The plugin creates a wingreek2utf8 object with one method, convert(), which you can apply to any selector.

jquery.wingreek-convert.js automatically combines some characters that are uncombined in WinGreek. There is a combine option to turn this off.

convert() will raise an error if it detects a character that is not included in the original character mapping of the font. You can set the ignore_invalid option to true to supress these errors.

jquery.wingreek-convert.js will generate characters not on this chart when it combines some glyphs, mostly capital letters and diacritial marks. For instance, WinGreek has no mapping for a capital alpha with rough breathing and acute accent. You would render that with two characters: “ (0x93), followed by A (0x41). When those characters occur by themselves (as they do here) they are converted into the equivalent Unicode characters 0x1fde and 0x0391, respectively. When they occur together, they will be combined into the single Unicode character 0x1f0d: “A.
